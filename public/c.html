Introduction to C Programming

C is a powerful general-purpose programming language. It is fast, portable, and available on all platforms.

Basic Syntax

The syntax of C programming is the set of rules that defines how a C program will be written and interpreted (both by the human reader and by the computer).


#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

Examples

Adding Two Numbers


#include <stdio.h>

int main() {
    int a, b, sum;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);
    sum = a + b;
    printf("Sum: %d", sum);
    return 0;
}
